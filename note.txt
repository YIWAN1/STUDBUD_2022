Steps to activate STUDBUD web page

Step1: # Install template to run by running the command:
npm install --save @templatejs/runtime

Step2: # Install the template compiler plugin by running the command:
npm install --save-dev parcel-plugin-template

（💠 This is a template plugin that makes it easier to implement interactive page displays and data changes）

Step3: Delete the node_modules and parcel-cache from the code
#delete node_modules
#delete parcel-cache

Step4: Install the SASS package by running the command:
npm install sass

Step5:  Start the blank web server using the command:
npm run start

Step6: Open https://localhost:8888 web server
(The web page is blank….No worries😉 )

Step7:  Run your server with the following command:
npm run dev

Step8： Return  https://localhost:8888 web server screen and press f14 to refresh the page

🎉 The website created is now right in front of you!
